PRIVACY POLICY

プライバシーポリシー

お客さまからお預かりする個人情報の取り扱いについて。

1 ／ 取得する情報

はいばら農園（以下「当農園」）は、お問い合わせフォームを通じて、以下の情報を取得します。

  • お名前
  • メールアドレス
  • お問い合わせ内容（ご希望の作物、数量、お届け先のご住所など、お客さまが任意で入力された情報）
2 ／ 利用目的

取得した個人情報は、以下の目的にのみ利用します。

  • お問い合わせへのご返信
  • 収穫時期や農園の状況に関するご案内（お客さまが希望された場合）

※ 当サイトでは商品のご注文・ご予約の受付は行っておりません。販売の方法・時期が決まり次第、本サイトでご案内いたします。

3 ／ 第三者提供

取得した個人情報を、お客さまの同意なく第三者に提供することはありません。

法令に基づく開示請求があった場合は、必要な範囲で開示することがあります。

4 ／ 利用ツール

本サイトでは以下の外部サービスを利用しています。

  • Google フォーム ／ お問い合わせフォームに利用。入力内容は Google 社のサーバーに保存されます。
  • Google Search Console ／ サイトの検索結果での表示状況を把握するために利用。個人を特定する情報は取得しません。
  • GitHub Pages ／ サイトの配信に利用。サーバーのアクセスログは GitHub 社の管理下で一時的に保存されます。

これらのサービスにおける個人情報の取り扱いについては、各社のプライバシーポリシーをご確認ください。

  • Google ／ https://policies.google.com/privacy
  • GitHub ／ GitHub Privacy Statement
5 ／ アクセス解析

本サイトでは現在、Cookie を使用したアクセス解析ツール（Google Analytics 等）は導入していません。

将来、利用状況の把握のためにアクセス解析ツールを導入する場合は、本ポリシーを改定したうえで以下の情報を記載します。

  • 利用するツール名と提供事業者
  • 取得するデータの種類（IPアドレス・ブラウザ情報・閲覧履歴など）
  • Cookie の利用有無と無効化の方法
  • データの利用目的と保管期間
6 ／ 情報の開示・訂正・削除

お客さまご自身の個人情報について、開示・訂正・削除をご希望の場合は、下記のお問い合わせ先までご連絡ください。本人確認のうえ、合理的な期間内に対応いたします。

7 ／ お問い合わせ先

個人情報の取り扱いに関するお問い合わせは、下記までお願いいたします。

haibara.farm@gmail.com

8 ／ 改定

本ポリシーの内容は、必要に応じて変更することがあります。変更後の内容は本ページに掲載した時点で効力を生じるものとします。
